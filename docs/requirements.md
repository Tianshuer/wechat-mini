# 天气伴旅小程序 - 需求文档

## 1. 项目概述

- **项目名称**: 天气伴旅
- **类型**: 微信小程序
- **技术栈**: Vue3 + TypeScript + Pinia + uni-app
- **定位**: 为用户提供城市天气查询、行程天气规划、收藏管理的工具型应用

---

## 2. 功能需求

### 2.1 首页（天气查询）

| 功能 | 描述 | 状态 |
|------|------|------|
| 城市搜索 | 输入城市名搜索，返回候选城市列表 | 已实现 |
| 实时天气展示 | 显示当前温度、天气文字、风向风速、湿度 | 已实现 |
| 天气图标 | 根据天气码展示对应的 SVG 图标 | 已实现 |
| 快捷收藏入口 | 展示用户收藏的城市列表（横向滚动） | 已实现 |
| 跳转行程天气 | 快捷入口跳转至行程天气页面 | 已实现 |
| 查看预报 | 跳转至天气预报页面 | 已实现 |
| 收藏城市 | 登录后可收藏当前城市 | 已实现 |
| 未登录提示 | 未登录时提示用户去登录 | 已实现 |

### 2.2 登录/注册

| 功能 | 描述 | 状态 |
|------|------|------|
| 手机号登录 | 输入手机号 + 短信验证码登录 | 已实现 |
| 发送验证码 | 前端倒计时防抖，后端发送短信 | 已实现 |
| 用户token持久化 | 登录成功后token存至storage | 已实现 |
| 微信登录 | 支持微信授权登录（code换token） | 已实现 |

### 2.3 行程天气

| 功能 | 描述 | 状态 |
|------|------|------|
| 日期范围选择 | 选择出行开始/结束日期 | 已实现 |
| 按日分配城市 | 在日期范围内每天指定一个城市 | 已实现 |
| 城市搜索选择 | 行程中每天可搜索并选择城市 | 已实现 |
| 行程保存 | 保存行程数据到本地 | 已实现 |
| 行程编辑 | 查看模式下可编辑行程 | 已实现 |
| 行程天气展示 | 查看行程中每天的天气情况 | 部分实现 |

### 2.4 天气预报

| 功能 | 描述 | 状态 |
|------|------|------|
| 天气预报列表 | 展示多天预报，每日一行 | 已实现 |
| 天气切换天数 | 支持 3/5/7/10 天预报切换 | 已实现 |
| 详细天气信息 | 显示温度区间、湿度、降水、风向风速 | 已实现 |

### 2.5 我的收藏

| 功能 | 描述 | 状态 |
|------|------|------|
| 收藏列表 | 展示用户收藏的城市列表 | 已实现 |
| 查看天气 | 点击收藏项查看该城市天气 | 已实现 |
| 删除收藏 | 删除某条收藏记录 | 已实现 |
| 登录限制 | 未登录时提示先登录 | 已实现 |

### 2.6 个人中心

| 功能 | 描述 | 状态 |
|------|------|------|
| 登录状态展示 | 显示登录/未登录状态 | 已实现 |
| 退出登录 | 清除token和用户信息 | 已实现 |

## 3. 页面结构

```
pages.json (共7个页面，3个tabBar页面)

TabBar 页面:
├── 首页 (pages/index/index)        # 首页/天气查询
├── 收藏 (pages/favorites/favorites) # 我的收藏
└── 我的  (pages/profile/profile)   # 个人中心

普通页面:
├── 登录   (pages/auth/login)
├── 注册   (pages/auth/register)
├── 行程天气 (pages/trip/trip)
└── 天气预报 (pages/forecast/forecast)
```

## 4. 数据模型

### 4.1 Store

| Store | 用途 |
|-------|------|
| weatherStore | 当前天气、预报、搜索结果、收藏列表 |
| userStore | 用户信息、token、登录状态 |

### 4.2 API 分类

| 模块 | 端点 |
|------|------|
| 认证 authApi | login, register, wechatLogin, sendSms, smsLogin |
| 天气 weatherApi | searchLocation, getCurrentWeather, getForecast, getTripWeather |
| 收藏 favoritesApi | getList, add, remove |
